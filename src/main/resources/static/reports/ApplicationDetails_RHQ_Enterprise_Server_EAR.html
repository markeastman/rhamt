Issue Detail: Seam 2 embedded library

Show Rule

The application has a Seam library embedded.
While official support for Seam 2.2 applications was dropped in JBoss EAP 6, it was still possible to configure dependencies for JSF 1.2 and Hibernate 3 to allow Seam 2.2 applications to run on that release.
